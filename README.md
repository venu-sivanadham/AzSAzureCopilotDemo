
# How to use

1. Open Azure Portal using this link: [MPAC Portal - Alternate Link](https://ms.portal.azure.com/?feature.azurepluginstore=true&feature.unifiedcopilotdebug=true&exp.copilotagents=true&feature.unifiedcopilot=true#home) .
1. Click on Copilot button on portal header.
1. Also make sure the advanced agent mode is enabled by clicking on the Agent icon just below copilot chat input text bar (The agent icon is next to the + button).
1. Click on `Test your plugin or agent` banner, a `Copilot testing` blade will open on left side of the Copilot chat.
1. Register Agent(s) by following one of the options below: 
    1. **Native Agents or Plugins:** Copy and paste the Restaurant [Agent/Plugin Manifest](./RestaurantAgent.json) json content in left playground and then click on Register button.
    2. **Integrated Agents:** Copy and paste the [Restaurant Integrated Agent](./Integrated_RestaurantAgent.json) json content in left playground and then click on Register button.
1. You will see Agent/Plugin successful registration messages in the Copilot chat.
1. Now you are all set to experiment with Restaurant copilot agent.
